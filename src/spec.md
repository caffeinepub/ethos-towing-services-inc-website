# Specification

## Summary
**Goal:** Add an ETR Everyday subscription membership section to showcase the $110 membership program with its benefits and integrate it into the main page flow.

**Planned changes:**
- Create ETRMembership component displaying "Ethos Towing & Recovery Everyday" (branded as "ETR Everyday") with $110 pricing
- Add seven feature cards: 24/7 roadside assistance, mobile battery service, flat tire service, emergency fuel delivery, lockout services, member discounts, and expert automotive advice
- Position the membership section between Services and ServiceArea components in the page layout
- Update chatbot responses to include ETR Everyday membership information

**User-visible outcome:** Users can view the ETR Everyday membership offering with clear pricing and benefits, and ask the chatbot about membership details.
